# 要件定義書

## プロダクト名
**「ZENをさがせ！」**

## コンセプト
大量の似た漢字の中に隠れた「禅」や「善」を探す、間違い探し系クソゲー。
「混沌の中にZENを見つける」という禅の本質をゲームで体現する。

---

## ゲーム概要

| 項目 | 内容 |
|------|------|
| ジャンル | 間違い探し / クソゲー |
| プラットフォーム | Webブラウザ（PC・スマホ対応） |
| プレイ人数 | 1人 |
| プレイ時間 | 1ステージ 30秒〜2分程度 |

---

## ステージ設計

### 1面：「禅を見つけよ」
- 背景文字：**弾**（たま）
- 隠れ文字：**禅**（ぜん）
- 難易度：中（弾と禅は形が似ている）

### 2面：「善を見つけよ」
- 背景文字：**悪**（あく）
- 隠れ文字：**善**（ぜん）
- 難易度：中（悪の中に善が1つ）

### 3面以降（拡張案）
| 面 | 背景文字 | 隠れ文字 | 読み |
|----|---------|---------|------|
| 3面 | 煩 | 禅 | 煩悩の中の禅 |
| 4面 | 戦 | 禅 | 戦の中の禅 |
| 5面 | 全 | 禅 | 全の中の禅（語呂合わせ） |

---

## ゲームメカニクス

### 基本ルール
1. 画面いっぱいに同じ漢字が敷き詰められる
2. その中に1文字だけ違う漢字が隠れている
3. プレイヤーがその文字をクリック/タップする
4. 正解 → 次のステージへ
5. 不正解 → 「それは煩悩です」と表示され続ける

### クソゲー要素（重要）
- 隠れ文字の位置はランダム
- 時間経過で文字がゆっくり動く（上級）
- 間違えるたびに文字が増える
- 正解してもすぐ次の面が来て休めない
- タイムアタックモード：制限時間内に何面クリアできるか

## AI活用ポイント（審査基準対策）

| 活用箇所 | 内容 |
|---------|------|
| ステージ自動生成 | AIが「似ている漢字ペア」を無限に生成する |
| 難易度調整 | プレイヤーの正解率をAIが分析し動的に難易度を変える |
| 煽りコメント | 不正解時にAIが禅語録風の煽り文句を生成する |
| エンディング | クリア後にAIが「あなたの悟り度」を診断して一言コメント |

### AI生成の漢字ペア例
```
（背景）鐘 → （隠れ）禅
（背景）弾 → （隠れ）禅
（背景）悪 → （隠れ）善
（背景）煩 → （隠れ）禅
（背景）戦 → （隠れ）禅
```

## 画面設計

### タイトル画面
- 黒背景
- 「ZENをさがせ！」タイトル
- STARTボタン

### ゲーム画面
- 黒背景 + 白文字でグリッド表示
- 上部：ステージ名（例：「禅を見つけよ」）
- 右上：タイマー・スコア
- グリッド：漢字がびっしり並ぶ
- クリックで判定

### 正解時
- エフェクト（光る・拡大）
- 「悟った！」テキスト表示
- 次のステージへ自動遷移

### 不正解時
- 「それは煩悩です」テキスト表示
- AIが生成した禅語録風の煽り文句

### クリア画面
- 「悟りスコア」表示
- AIによる一言診断コメント
- SNSシェアボタン
